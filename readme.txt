Display recent game information on your blog.

== Description ==

**Ever get tired of having to input game information any time you want to display its information on your blog. Harness the power of one of the leading Game databases, IGDB.com to display any game you want.** 

Display game information like:

*Game Boxart
*Game Title
*Release Date
*Developers and Publishers
*Platforms this game is avalaible on.

== Installation ==

Insalation is simple.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Visit [IGDB.com Mashape Page](https://market.mashape.com/igdbcom/internet-game-database) and request a key by registering for the site.
1. In the settings section under the WPIGDB add the newly acquired API key and save.
1. When you want to add a game, just click on the IGDB.com button on your posts text editor. And search for the game you want to add, then hit select.


== Frequently Asked Questions ==

= How do I get a api key? =

You visit [IGDB.com Mashape Page](https://market.mashape.com/igdbcom/internet-game-database) and request a key by registering for the site.
 
= How do I add a game to my post? =

In the text editor for the post you should see the IGDB.com icon. When you click it, a popup will show with a search bar. Start entering the name of the game. Once the game you want to add appears, click the select link and it will add it to the post.
